Zombie Lane Defense - Map Editor Testing

Map Editor

Create and edit maps for Zombie Lane Defense. Place enemies, obstacles, and bonuses on lanes.

Quick Test Steps:

  1. Click "Create New Map" and enter a name and length
  2. Click on lanes to place Normal Zombies
  3. Double-click on an entity to edit its properties
  4. Click and drag entities to reposition them
  5. Click "Save to File" to download your map

Map Viewer

View and navigate maps created in the Map Editor. Test how your maps will appear in-game.

Quick Test Steps:

  1. Click "Load Map" and select a map file
  2. Use the scroll buttons to navigate through the map
  3. Verify that all entities appear correctly
